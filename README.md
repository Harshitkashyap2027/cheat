# B25 CHEATS – Fast Emote System

A fast emote-sending tool for **Free Fire** that lets you trigger any in-game emote on demand — including on **Social Island**.

---

## 🏝️ How to Show Emotes on Social Island

Social Island is a special lobby mode in Free Fire where players can hang out and interact. Follow the steps below to send emotes there using this tool.

### Step 1 – Open the Tool
Open **`B25 CHEATS - Fast Emote System.html`** in your browser.

### Step 2 – Switch to Social Island Mode
In the **Control Panel** section, click the **"Social Island"** button (the one with the beach umbrella icon ☂️).  
> The **Team Code** field will automatically hide — it is not needed for Social Island.

### Step 3 – Select Your Server
Choose the server that matches your Free Fire account from the **Server** dropdown (e.g., India, Indonesia, Brazil, etc.).

### Step 4 – Enter UIDs
Fill in the **UID** fields with the Free Fire user IDs of the players you want to send emotes to:
- **UID 1** – Required (your own UID or the target player's UID)
- **UID 2 / 3 / 4** – Optional (add more players if needed)

### Step 5 – (Optional) Enable Extra Options
- ✅ **Auto Leave** – Automatically leaves after the emote is sent.
- ⚡ **Lag Mode** – Enables lag mode for the session.

### Step 6 – Choose and Send an Emote
Scroll down to the **Emotes** grid (344 emotes available). Click the **Send** button on any emote card to trigger it.

The **Response** box at the top will show you the result of the request.

## 🆚 Team / Group Mode vs Social Island Mode

| Feature        | Team / Group Mode        | Social Island Mode     |
|----------------|--------------------------|------------------------|
| Team Code      | ✅ Required               | ❌ Not needed          |
| UIDs           | ✅ Required (1–4)         | ✅ Required (1–4)      |
| Use case       | Squad lobby / team rooms | Social Island lobby    |

---

## 📋 Notes
- Make sure you are **inside Social Island** in-game before sending the emote.
- The UIDs must belong to players currently present on Social Island for the emote to appear.
- This tool sends requests to the B25 Cheats backend API (`/send_emote_social` endpoint for Social Island).
